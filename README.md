# chip-8

> [!WARNING]
> Please note that this project is a work in progress.

A simple CHIP-8 interpreter in C.

## Build

Just run make. It will create `chip-8-run` in `build` folder.

```
make
```

## Usage

```
chip-8-run [PATH TO YOUR PROGRAM]
```

For example:

```
chip-8-run ibm-logo-ch.8
```
